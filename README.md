\# Odor-Free Restoration Quote Builder



Custom WordPress plugin for creating and managing service quotes for Odor-Free Restoration.



Sales users can create client quotes, calculate labor and material totals, manage saved quotes, and generate shareable PDF documents.



\## Main Features



\* Private quote builder for logged-in users

\* Automatically generated unique quote numbers

\* Custom database tables for quotes and quote line items

\* Client and billing contact information

\* Separate labor and materials sections

\* Custom service and material line items

\* Automatic line-item, subtotal, tax, and total calculations

\* Quote preview and print view

\* PDF download and email delivery

\* Quote statuses:



&#x20; \* Draft

&#x20; \* Generated

&#x20; \* Revised

&#x20; \* Sent

&#x20; \* Closed

&#x20; \* Deleted

\* Authorized users can search, view, edit, and manage all quotes



\## Shortcode



Add the quote builder to a WordPress page using:



```text

\[odorfree\_quote\_builder]

```



\## Install Location



Install the plugin in:



```text

wp-content/plugins/odorfree-quote-builder

```



\## Notes



This plugin is designed specifically for the Odor-Free Restoration service-quote workflow.
